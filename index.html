Maze Generator

This project is implemented in Java and must be run on your computer.

To run the maze generator and view output, use a Java IDE like IntelliJ or VS Code, or compile via terminal:

javac Main.java
java Main

GitHub Pages cannot execute Java code directly in browser.

If you want to visualize mazes on this site using JavaScript instead, I can help convert your logic into web-based code. Just say the word!
